# Dino-Game-AI

Simple Dino game using p5.js, Genetic Algorithm and TensorFlow.js

<br>

# Getting Started

1. Clone the repository

```bash
git clone https://github.com/sherwyn11/Dino-Game-AI.git
```

2. Open index.html in your browser

<br>

# Dependencies

<ul>
  <li>HTML5</li>
  <li>p5.js</li>
  <li>TensorFlow.js</li>
</ul>
